Desigualdad de Korovkin

Si $x_{1}, x_{2}, \ldots, x_{n} \in \mathbb{R}_{+}$ satisfacen $x_{1} \cdots x_{n} = 1$, entonces $$x_{1} + x_{2} + \cdots + x_{n} \geq n.$$ La igualdad se da si y sólo si $x_{1} = x_{2} = \ldots = x_{n} = 1$

Desigualdad de Korovkin

Si $x \in [0,1)$, entonces $$1-x \leq e^{-x}$$

Desigualdad media Aritmética - media Geométrica

Si $x_{1}, x_{2}, \ldots, x_{n} \in \mathbb{R}_{+}$, entonces $$\sqrt{x_{1} \cdot x_{2} \cdots x_{n}} \leq \frac{x_{1} + x_{2} + \cdots + x_{n}}{n}.$$ La igualdad se da si y sólo si $x_{1} = x_{2} = \ldots = x_{n}$

Desigualdad media Cuadrática - media Aritmética

Si $x_{1}, x_{2}, \ldots, x_{n} \in \mathbb{R}_{+}$, entonces $$ \frac{x_{1} + x_{2} + \cdots + x_{n}}{n} \leq \sqrt{\frac{x_{1}^2 + x_{2}^2 + \cdots + x_{n}^2}{n}} .$$ La igualdad se da si y sólo si $x_{1} = x_{2} = \ldots = x_{n}$

Desigualdad media Armónica - media Aritmética - media Geométrica

Si $x_{1}, x_{2}, \ldots, x_{n} \in \mathbb{R}_{+}$, entonces $$\frac{n}{\displaystyle{\frac{1}{x_{1}} + \frac{1}{x_{2}} + \cdots + \frac{1}{x_{n} }}} \leq \sqrt[n]{x_{1} \cdot x_{2} \cdots x_{n}} \leq \frac{x_{1} + x_{2} + \cdots + x_{n}}{n}.$$ La igualdad se da si y sólo si $x_{1} = x_{2} = \ldots = x_{n}$

Desigualdad de Cauchy-Schwarz

Si $x_{1}, x_{2}, \ldots, x_{n}, y_{1}, y_{2}, \ldots, y_{n} \in \mathbb{R}$, entonces $$\left| \sum_{i=1}^{n} x_{i} y_{i} \right| \leq \left( \sum_{i=1}^{n} x_{i}^{2} \right)^{\frac{1}{2}} \left( \sum_{i=1}^{n} y_{i}^{2} \right)^{\frac{1}{2}} $$

Lema de Titu

Si $x_{1}, x_{2}, \ldots, x_{n}, y_{1}, y_{2}, \ldots, y_{n} \in \mathbb{R}_{+}$, entonces $$\frac{\displaystyle{\left(\sum_{i=1}^{n} x_{i} \right)^{2}}}{\displaystyle{\sum_{i=1}^{n} y_{i}} } \leq \sum_{i=1}^{n} \frac{x_{i}^{2}}{y_{i}} $$
